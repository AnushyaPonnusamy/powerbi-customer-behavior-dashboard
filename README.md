# Power BI Sales Dashboard

## Overview
This project is an interactive customer behavior  dashboard created using Power BI to analyze business performance and key metrics.

## Tools Used
- Power BI
- DAX
- CSV

## Features
- KPI cards (Revenue, Profit, Sales)
- Filters and slicers
- Trend analysis
- Category-wise insights
